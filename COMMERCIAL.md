HoldType is a source-available macOS voice typing application.

The source code is available so users can inspect how the app works, learn from
it, build it for personal or internal use, including inside a company, and
contribute improvements.

The license does not allow using this code to build, distribute, sell, host, or
provide a competing product or service during the license period.

Examples of uses that require explicit permission:

- distributing a rebranded HoldType build;
- selling a modified HoldType build;
- publishing a competing voice typing or dictation app based on this code;
- using the HoldType name, icon, logo, screenshots, or domain in a fork;
- packaging HoldType as part of a commercial product without permission.

Personal and internal use are allowed. You may build HoldType from source and
run your own build, including for security review, internal evaluation, or
private daily use.

The restriction is about competing products or services, commercial
repackaging, and HoldType brand use, not normal private or internal builds.

For commercial licensing or permission, open a GitHub issue or contact the
project owner.
